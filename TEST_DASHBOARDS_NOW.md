## ğŸ› Troubleshooting

### Dashboard not loading?
- Check console for errors (F12)
- Verify you clicked "Submit" on auth form
- Try refreshing the page

### Upload not working?
- Make sure you selected an image file
- Check file size (should be < 10MB)
- Try a different image format (PNG, JPG)

### AI verification stuck?
- Wait 2 seconds for simulation
- Check console for errors
- Refresh and try again

### Sidebar collapsed on desktop?
- This is normal on mobile (<768px)
- Resize browser window to full width
- Sidebar will expand automatically
